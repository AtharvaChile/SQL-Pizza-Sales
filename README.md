# Pizza-Sales
## 🍕 Pizza Sales Analysis using SQL
### 📌 Project Overview

This project analyzes a pizza sales dataset using SQL to extract meaningful business insights. The goal is to perform data analysis on sales, revenue, customer ordering behavior, and product performance using structured queries.
### 📊 Business Problems Solved

The following key business questions were solved using SQL:

1.Total number of orders

2.Total revenue generated

3.Highest priced pizza

4.Most common pizza size ordered

5.Top 5 most ordered pizzas

6.Total quantity ordered per category

7.Distribution of orders by hour

8.Category-wise pizza distribution

9.Average number of pizzas ordered per day

10.Percentage contribution of each pizza category to revenue

11.Top 3 pizzas by revenue

12.Cumulative revenue over time

13.Revenue generated per category

### 📈 Key Insights

Identified the best-selling pizza categories

Determined peak order hours

Calculated total and category-wise revenue

Found high-performing pizza sizes

Analyzed customer ordering patterns
